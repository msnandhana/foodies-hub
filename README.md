# Foodies Hub Restaurant Website

Foodies Hub is a web application that allows users to browse food items, add them to cart, and place orders through a simple and user-friendly interface.

## Features
- Browse food menu (Veg & Non-Veg items)
- Add items to cart
- Checkout and place orders
- Order management pages
- Responsive user interface
- Simple and user-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB (if used)
